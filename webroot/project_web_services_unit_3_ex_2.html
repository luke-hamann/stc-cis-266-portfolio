Products SOAP API with Database Emulator

This application emulates a database of products and provides a SOAP API for accessing them. Naturally, this includes the business classes and the SOAP web service itself. However, it also includes JSP pages for testing the business classes as well as a SOAP web service test client. The program was completed as an assignment to fulfill the requirements of a web services course.

The technologies include:

  • Java
  • JSP
  • IBM WebSphere
  • SOAP
  • WSDL
  • XSD
