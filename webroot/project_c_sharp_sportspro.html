SportsPro Technical Support Website

This is a website for managing technical support tickets. It includes pages for managing products, technicians, customers, incidents, and registrations. Some functionality is login-protected. The application was a group project for an ASP.NET Core MVC course.

My responsibilities included:

  • Writing most components relating to the product management pages
  • Writing some components relating to validation on the customer management pages
  • Refactoring the data layer to use a repository pattern (and make it easier to unit test)
  • Implementing role-based administrative access

The technologies include:

  • C#
  • ASP.NET Core MVC
  • Entity Framework Core
  • ASP.NET Core Identity
  • Razor pages
  • Bootstrap
  • jQuery
