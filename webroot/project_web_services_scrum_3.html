Car Dealership Data Website

This application allows a user to manipulate data relating to a car dealership, including cars, customers, and salespersons. AJAJ-enhanced forms perform CRUD operations without page reloads. It also includes testers for viewing all records as well as the output of JSON endpoints. The website was completed as a group project for a web services class.

The technologies include:

  • PHP
  • MySQL
  • JSON
  • AJAJ
