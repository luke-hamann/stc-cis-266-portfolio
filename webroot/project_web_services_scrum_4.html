Cats SOAP API

This application provides an "echo" API for inputting and outputting cat information, accessible through both a web application and a SOAP web service. The API accepts information relevant to a cat: name, color, and weight. It then constructs a cat object and returns the data in an appropriately formatted manner. The cat entity class, API class, HTML input form, and JSP processing page were written from scratch. We then used IBM WebSphere to generate a SOAP web service and test client based on our classes. This program was created as a group project to fulfill the requirements of a web service course.

The technologies include:

  • Java
  • JSP
  • IBM WebSphere
  • SOAP
  • WSDL
  • XSD
