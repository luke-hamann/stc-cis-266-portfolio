Product Line Item Manager

This is a Java desktop application for managing a database of products and line items. It provides all CRUD functionality for both products and line items as well as a search capability. Data is stored in a SQLite database. This application was written as a bonus assignment for an advanced level Java class. I developed this application individually by watching and "cloning" a user interface demonstration of a potential solution provided by our professor.

The technologies include:

  • Java
  • JavaFX
  • SQLite
